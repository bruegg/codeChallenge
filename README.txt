How to run number guessing game:
 - cd {workspace}/src/main/guessinggame
 - javac GuessingGame.java
 - cd ../..
 - java main.guessinggame.GuessingGame


How to run ChessGame
 - cd {workspace}/src/main/chessgame
 - javac *.java
 - cd ../..
 - java main.chessgame.ChessGame {piece} {position}
 ex: java main.chessgame.ChessGame knight d2
